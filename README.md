# REST vs SOAP - Postman Practice

This repo contains:
- Postman Collection: `postman/Practica-RESTvsSOAP.postman_collection.json`
- Postman Environment example (sanitized): `postman/REST_vs_SOAP_-_Local.postman_environment.example.json`

## How to use
1. Import the collection JSON into Postman.
2. Import the environment example and fill your local variables (baseUrl, user, etc.).
3. Run the requests in order (Login -> CRUD -> SOAP).
